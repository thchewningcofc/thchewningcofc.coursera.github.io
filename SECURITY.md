# Security Policy

## Supported Versions

This repository is actively maintained on the default branch. Security fixes are applied there first.

| Version | Supported |
| --- | --- |
| Default branch | ✅ |
| Older commits/releases | ❌ |

## Reporting a Vulnerability

If you discover a security issue, please **do not** open a public GitHub issue with exploit details.

Instead, report privately by email:

- **Email:** tylerharrisonchewning@gmail.com
- **Subject:** `Security vulnerability report: thchewningcofc.coursera.github.io`

Please include:

1. A clear description of the issue and impact.
2. Steps to reproduce.
3. A proof-of-concept (if available).
4. Suggested mitigation (optional).

## Response Expectations

- Initial acknowledgment: within **3 business days**.
- Status update after triage: within **7 business days**.
- Remediation timeline depends on severity and complexity.

Thank you for helping improve the security of this project.
